## Data Structure

### 1. String

- StringBuffer 线程安全
- StringBuilder 线程不安全，单线程下效率高一些，一般情况使用

### 2. LinkedList

- 线性表
  - 链表 内存不连续，由指针连接；查询慢O(n);增删快O(1)
    - 单链表
    - 双链表
    - 循环链表
  - 顺序表 内存连续；查询快O(1)；增删慢O(n)
